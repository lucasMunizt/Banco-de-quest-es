Projeto Livraria: Desenvolvimento de REST APIs com Spring Boot

Gabarito

Você foi designado para criar um sistema de gerenciamento para uma livraria. O sistema deve permitir aos usuários visualizar, a dicionar, atualizar e excluir livros. Cada livro possui um título, autor, ano de publicação e uma categoria. Além disso, os usuários devem ser capazes de consultar a lista de categorias disponíveis.

Exemplos de APIs:

  • GET /api/livros: Recupera a lista de todos os livros.
  • GET /api/livros/{id}: Recupera os detalhes de um livro específico por ID.
  • POST /api/livros: Adiciona um novo livro.
  • PUT /api/livros/{id}: Atualiza as informações de um livro existente por ID.
  • DELETE /api/livros/{id}: Exclui um livro por ID.
  • GET /api/categorias: Recupera a lista de todas as categorias disponíveis.
  • GET /api/categorias/{id}: Recupera os detalhes de uma categoria específica por ID.
  • POST /api/categorias: Adiciona uma nova categoria.
  • PUT /api/categorias/{id}: Atualiza as informações de uma categoria existente por ID.
  • DELETE /api/categorias/{id}: Exclui uma categoria por ID.
